# lab5
## Лабораторная работа 5: Agile и Scrum

## Описание проекта
Проект представляет собой веб-приложение для управления задачами по методологии Scrum. Приложение позволяет создавать задачи, отслеживать их статус, планировать спринты и анализировать прогресс команды. Интерфейс интуитивно понятен и адаптирован для ежедневного использования. Цель проекта - повысить эффективность командной работы над IT-проектами.

## Sprint Retrospective

**Что прошло хорошо:**
1. Эффективное распределение задач между разработчиками
2. Регулярные ежедневные стендапы (Daily Scrum)
3. Быстрое решение технических проблем с базой данных

5. Своевременное выполнение первых двух задач

2. Задержки из-за проблем с настройкой окружения разработки

4. Улучшить документацию критериев готовности (DoD)
